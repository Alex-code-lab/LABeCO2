
# Installation et utilisation de Homebrew

## Installation de Homebrew
1. Ouvrez un terminal sur votre Mac.
2. Copiez-collez la commande suivante pour installer Homebrew :
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```
3. Suivez les instructions affichées à l'écran pour terminer l'installation.
4. Une fois installé, ajoutez Homebrew à votre `PATH` en exécutant :
   ```bash
   echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zshrc
   eval "$(/opt/homebrew/bin/brew shellenv)"
   ```
5. Vérifiez que Homebrew est correctement installé :
   ```bash
   brew --version
   ```

## Utilisation de Homebrew
### Installer des paquets
Pour installer un logiciel ou une bibliothèque :
```bash
brew install nom_du_paquet
```
Pip install pyside6
brew install qt@6
pip install tables
brew install python@3.13 pyqt qt
Pip install matplotlib adjustText pandas pyinstaller


Pip install -r requirements-brew.txt

### Désinstaller des paquets
Pour désinstaller un logiciel ou une bibliothèque :
```bash
brew uninstall nom_du_paquet
```

### Mettre à jour Homebrew et ses paquets
1. Mettre à jour Homebrew lui-même :
   ```bash
   brew update
   ```
2. Mettre à jour tous les paquets installés :
   ```bash
   brew upgrade
   ```

### Créer des environnements Python avec Homebrew
1. Installez Python via Homebrew (si ce n'est pas déjà fait) :
   ```bash
   brew install python
   ```
2. Créez un environnement virtuel dans le dossier de votre projet :
   ```bash
   python3 -m venv nom_environnement
   ```
3. Activez l'environnement virtuel :
   ```bash
   source nom_environnement/bin/activate
   ```
4. Installez des bibliothèques spécifiques dans cet environnement :
   ```bash
   pip install nom_du_paquet
   ```
5. Désactivez l'environnement virtuel :
   ```bash
   deactivate
   ```
